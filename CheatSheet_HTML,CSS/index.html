CSS Commonly Used Properties

Name Description Example
font-family Font style font-family: Garamond
font-size Size of the element font-size:12px
color Color of the element color:blue
background-color Background color of the page background-color: orange
font-weight To make the dont bold or normal font-weight: bold
text align Alligning the text to be center, left or right text-align:center
opacity Alter the visibilty of the element opacity:0.5
background-image Background image for the page background-image:url('address')
